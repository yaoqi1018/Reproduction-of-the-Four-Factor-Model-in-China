# 中国四因子模型（CH4）—— 数据清洗方法与最终结果说明

---

## 一、项目概述

本项目基于 CSMAR 数据库，按照中国四因子模型（CH4）标准方法，构建 A 股市场的市场因子（MKT）、规模因子（SMB）、价值因子（VMG）和情绪因子（PMO），并将自建因子与官方因子进行对比验证，最后对沪深300指数进行因子调整的 Alpha 分析。

- **时间范围**：2011-07 ~ 2026-04（约 178 个月）
- **股票池**：全部 A 股（剔除 ST、新股、停牌股后每月约 800~1000 只有效样本）

---

## 三、数据清洗方法

### 3.1 个股回报率清洗（preprocess.py → `preprocess_returns`）

以下清洗步骤按顺序执行，每步均统计筛除数量：

#### 步骤1：剔除上市不足6个月的新股
- **原因**：新股上市初期波动异常剧烈、股价行为不稳定，不适合纳入因子计算
- **方法**：计算 `trdmnt（月份）` 与 `listdt（上市日期）` 的月份差，小于 6 的剔除
- **配置参数**：`MIN_LISTED_MONTHS = 6`

#### 步骤2：剔除月内停牌过久的股票
- **原因**：月内大部分时间停牌的股票该月没有真实交易价格，收益率失真
- **方法**：逐月逐只统计停牌天数，停牌天数 > 15 天则当月剔除
- **停牌天数算法**：从 `停复牌信息.csv` 的停牌起始日和恢复日，按跨越的每个自然月分解，计算在该月内实际停牌的天数
- **配置参数**：`MIN_SUSP_DAYS = 15`

#### 步骤3：剔除 ST / *ST 股票
- **原因**：ST 股存在财务异常、退市风险等特殊因素，收益分布与其他股票系统性地不同
- **方法**：在公司名称中匹配 `ST` 或 `*ST` 关键字
- **配置参数**：`ST_FLAG = True`

#### 步骤4：市值单位换算
- **原因**：CSMAR 中 `Msmvttl`（总市值）和 `Msmvosd`（流通市值）字段单位为**千元**，而资产负债表的账面价值单位为**元**
- **方法**：在加载阶段统一乘以 1000 转为元
- **配置参数**：`MSMVTTL_SCALE = 1000`

#### 清洗结果
- 原始记录约 **50~70 万条**（全样本）
- 剔除项：上市不足6个月（少量）、月内停牌>15天（少量）、ST/*ST股（少量）
- 最终保留率约 **96%~97%**

---

### 3.2 财务报表数据处理

#### 资产负债表（取年报）
- 仅保留 `accper` 为 **12月31日** 的年报数据
- 仅保留**合并报表**（`typrep == "A"`），排除母公司报表
- 剔除股东权益 ≤ 0 的无效记录
- 每只股票每年取最后一条（去重）

#### 利润表（取年报）
- 同上：仅保留 12月31日年报、合并报表
- 提取**归母净利润**（`B002000101`）和**营业利润**（`B001100000`）
- 归母净利润为空则剔除

---

### 3.3 无风险利率处理

- CSMAR 中无风险利率为**百分比形式**（如 `0.2263` 表示 0.2263%）
- **除以 100** 转为小数形式（`0.002263`），与个股收益率单位一致
- 取每月第一条记录作为当月无风险利率

---

### 3.4 EP 数据处理

EP（盈利/价格比）用于价值因子 VMG 的分组：

- **时序匹配**：遵循 Fama-French 标准方法。t 年 7 月 ~ t+1 年 6 月，使用 **t-1 年12月年报**的归母净利润，除以 **t-1 年12月的总市值**
- **正向筛选**：仅保留归母净利润 > 0 且 EP > 0 的股票（亏损股不参与价值排序）
- **缩尾处理**：每月对 EP 值做 **1%/99% 分位数缩尾**，消除极端值干扰

---

### 3.5 异常换手率计算

情绪因子 PMO 基于**异常换手率**分组：

- **异常换手率定义**：当月日均换手率 ÷ 过去 12 个月日均换手率均值
- **最低历史要求**：至少需要过去 6 个月的换手率数据（`PM_MIN_TURN_MONTHS = 6`）
- 无限值和空值均剔除
- 计算 PMO 时额外**剔除市值最小 30% 的"壳股"**（`PM_SHELL_PCT = 0.30`），防止壳股投机干扰情绪因子度量

---

## 四、因子构建方法

### 4.1 市场因子 MKT

```
MKT = 全A股流通市值加权平均月收益率 − 月化无风险利率
```

- 以每只股票的**流通市值**为权重计算加权平均收益
- 扣除当月无风险利率得到市场风险溢价

### 4.2 规模因子 SMB（调整后）

SMB 采用**双重中性化**方法，对 EP 效应和换手率效应各做一次中性化后取平均：

```
SMB = (EP中性SMB + 换手率中性SMB) / 2
```

#### EP中性 SMB
- 每月按总市值中位数分 **S（小盘）/ B（大盘）** 两组
- 按 EP 分 **H（高EP，前30%）/ M（中EP，中40%）/ L（低EP，后30%）** 三组
- 形成 2×3 = 6 个组合，流通市值加权计算各组合月收益
- `SMB_EP = (SH+SM+SL)/3 − (BH+BM+BL)/3`

#### 换手率中性 SMB
- 先剔除市值最小30%壳股
- 每月按总市值中位数分 S / B
- 按异常换手率分 **L（低换手，前30%）/ M（中换手，中40%）/ H（高换手，后30%）**
- 6 个组合流通市值加权
- `SMB_turnover = (SL+SM+SH)/3 − (BL+BM+BH)/3`

### 4.3 价值因子 VMG

```
VMG = (SH+BH)/2 − (SL+BL)/2
```

- 基于 Size × EP 的 2×3 分组（与 EP中性SMB 共用分组）
- 做多高 EP 组合、做空低 EP 组合，规模维度取平均以中性化规模效应

### 4.4 情绪因子 PMO

```
PMO = (SL+BL)/2 − (SH+BH)/2
```

- 基于 Size × Abnormal Turnover 的 2×3 分组（与换手率中性SMB 共用分组）
- 做多低异常换手率组合、做空高异常换手率组合
- **经济学含义**：低换手率代表投资者关注度低、情绪冷静，高换手率代表投机情绪高涨；PMO > 0 表示"冷静股"跑赢"热门股"

## 六、沪深300 Alpha 分析

### 6.1 回归模型

将沪深300指数视为一个待评估的投资组合：

```
R_CSI300 − Rf = α + β1·MKT + β2·SMB + β3·VMG + β4·PMO + ε
```

### 6.2 回归结果

| 指标 | 数值 | 
|------|------|
| **Alpha（月化）** | 见 `output/csi300_alpha_regression.csv` |
| **Alpha（年化）** | 月化 × 12 |
| **R²** | 见完整回归输出 |

| 因子暴露 β | 数值 | 方向含义 |
|:---:|:------:|------|
| MKT | 见输出 | 沪深300对市场整体的敏感度（预期接近 1.0） |
| SMB | 见输出 | 负值表示沪深300偏向大盘股 |
| VMG | 见输出 | 正值表示偏向价值股 |
| PMO | 见输出 | 正值表示偏向低换手率股 |

> 注：完整的回归系数、t 值、p 值、F 统计量及诊断图详见 `output/csi300_alpha_analysis.png` 和 `output/csi300_alpha_regression.csv`。

### 6.3 关键结论

1. **模型拟合度**：R² 说明四因子能否有效解释沪深300的收益波动
2. **Alpha 显著性**：如果 Alpha 显著为负，说明沪深300在样本期内系统性跑输因子模型的预测水平
3. **风格暴露**：SMB 载荷为负确认沪深300的大盘属性，MKT 载荷接近 1.0 说明与市场高度同步
