# 中国四因子模型（CH-4）复现项目总结
本项目复现 Liu 等（2019）中国A股四因子模型（CH-4），包含市场因子 MKT、规模因子 SMB、价值因子 HML、换手率情绪因子 PMO；完成因子构造、描述统计、与CSMAR官方因子相关性对比，并对沪深300指数做风格暴露与Alpha归因分析。

## 一、数据说明
| 数据名称 | 来源 | 时间范围 | 核心字段 |
|----------|------|:---:|------|
| 月个股回报率 | CSMAR | 2010-01~2026-04 | 股票代码、月份、月收益率、总市值、流通市值 |
| 无风险利率 | CSMAR | 2010-01~2026-04 | 月份、月化无风险利率 |
| 资产负债表 | CSMAR | 2010~2026 | 股票代码、会计期间、股东权益 |
| 公司基本情况 | CSMAR | — | 股票代码、上市日期、行业 |
| 停复牌信息 | CSMAR | 2010~2026 | 股票代码、停牌日期、复牌日期 |
| 官方Carhart四因子 | CSMAR | 2010-01~2026-04 | 市场风险溢价、SMB、HML、UMD |
| 指数日交易数据 | CSMAR | 2010-01~2026-04 | 上证指数、沪深300等每日收益率 |

## 二、数据清洗规则
1. **剔除新股**：上市不足6个月的股票不参与因子计算，规避上市初期异常波动。
2. **剔除停牌股**：单月停牌超过15个交易日的股票，当月剔除样本池。
3. **剔除ST股**：剔除ST、*ST类财务异常个股，保证样本有效性。
4. **量纲统一**：CSMAR市值单位为千元、账面价值单位为元，统一换算为同口径单位。
5. **利率换算**：原始无风险利率为百分比形式，除以100转为小数参与回归。

最终样本保留 **96.4%** 原始数据，每月有效个股数量稳定在2000~5000只。

## 三、因子描述统计
| 因子 | 月均值 | 年化均值 | t值 | 解读 |
|------|:---:|:---:|:---:|------|
| MKT | +1.53% | +18.4% | +3.31 | A股市场存在显著正向风险溢价 |
| SMB | −1.24% | −14.9% | −3.70 | 样本区间内小盘股显著跑输大盘股 |
| HML | −0.30% | −3.6% | −0.90 | 价值股小幅跑输成长股，效应统计不显著 |
| PMO | +0.62% | +7.4% | +1.50 | 换手率情绪因子收益为正，但未达到显著水平 |

## 四、与CSMAR官方因子相关性
| 因子 | 相关系数 |
|------|:---:|
| MKT | 0.78 |
| SMB | 0.64 |
| HML | 0.09 |
| PMO | 0.56 |

### HML低相关性原因分析
1. CSMAR官方价值因子大概率采用**EP盈利价比**，而非传统BM账面市值比；
2. 账面价值统计口径差异（合并报表/母公司报表）；
3. 分组断点、市值加权方式、样本筛选规则存在细微差异。

## 五、沪深300指数Alpha归因分析
### 回归模型
沪深300超额收益 = α + β₁×MKT + β₂×SMB + β₃×HML + β₄×PMO + 残差

### 回归整体结果
| 指标 | 数值 |
|------|------|
| 月化Alpha | -0.004238 |
| 年化Alpha | -5.0861% |
| Alpha t值 | -2.3282 |
| 拟合优度R² | 96.24% |

### 因子风格暴露
| 因子暴露（β） | 数值 | 解读 |
|:---:|:---:|:----:|
| MKT | 0.99 | 沪深300走势与A股大盘高度同步 |
| SMB | −0.28 | 显著偏向大盘蓝筹风格 |
| HML | +0.08 | 小幅偏向价值风格 |
| PMO | −0.07 | 轻微偏向低换手率、反向收益特征 |

### 模型结论
1. 沪深300存在**显著负Alpha**：控制四因子风格后，指数长期收益低于模型理论预期；
2. 模型拟合效果优异，R²达96.24%，四因子可绝大部分解释沪深300收益波动；
3. 风格特征清晰：沪深300核心属性为**大盘蓝筹+轻度价值**，无明显动量趋势。
