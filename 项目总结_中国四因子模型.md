### 数据清理

1. **剔除新股**：上市不足6个月的股票不参与计算（刚上市波动太大）
2. **剔除停牌股**：一个月内停牌超过15天的股票当月不参与（无法交易）
3. **剔除ST股**：名称中带ST或*ST的股票不参与（财务异常）
4. **市值单位统一**：CSMAR数据中市值单位是"千元"，账面价值单位是"元"，统一换算
5. **无风险利率换算**：CSMAR的利率是百分比形式（如0.1856表示0.1856%），除以100转小数

最终保留了 **96.4%** 的原始数据，每月约有2000~5000只有效股票。

## 沪深300 Alpha 分析

### 回归模型

用整个沪深300指数作为一个"投资组合"，看看它的收益能不能被四因子完全解释：

```
沪深300超额收益 = α + β1×MKT + β2×SMB + β3×HML + β4×PMO + 残差
```

### 回归结果

| 指标 | 数值 | 解读 |
|------|------|------|
| **Alpha（月化）** | **−1.51%** | 每月跑输因子预测1.51% |
| **Alpha（年化）** | **−18.09%** | 年化跑输约18% |
| **Alpha t值** | −12.85 | 统计学上非常显著(p≈0)，不是偶然 |
| **R²** | **94.7%** | 四因子能解释沪深300 94.7%的收益波动 |

| 因子暴露（β） | 数值 | 解读 |
|:---:|:---:|------|
| MKT | **+0.99** | 沪深300几乎和大盘完全同步 |
| SMB | **−0.28** | 沪深300偏向大盘股（预期之中） |
| HML | **+0.08** | 沪深300略偏价值股 |
| PMO | **−0.07** | 沪深300略偏"输家"股 |

### 6.3 这意味着什么？

1. **沪深300有显著的负Alpha**：在2011-2026年期间，沪深300的表现系统性低于四因子模型预测的水平。即使考虑了市场涨跌、大小盘风格、价值成长风格和动量效应，沪深300每年仍然跑输约18%。

2. **模型拟合非常好（R²=95%）**：四个因子几乎完全捕捉了沪深300收益的变化，说明模型设定是合理的。

3. **沪深300是"大盘+略偏价值"指数**：SMB载荷为负(−0.28)说明它天然偏向大公司，MKT载荷接近1.0说明它和市场同步。
